- Koa， bluebird, request, ejs, lodash, heredoc, raw-body, sha1，xml2js

### 功能列表

- 自动回复(程序生成)
- js sdk的接入
    * 语音搜索
    * 扫二维码，拍照，相册等接口对接
- 回复的设定：文案, 图文, 图片、视频素材(永久,临时), mp3, 文章列表, 用户列表...
- 菜单的设定
- 等等

### 开发相关

- 网站，公众号可以共用同一个后台
- 本地调试环境与微信远程通信，需要进行网络代理
    * QQ浏览器, ngrock等
- 公众号交互流程
    * 公众号开发需要与公众号后台有交互，需要在后台网页进行配置：通信域名地址，js sdk授权地址等
- 公众号相关：
    * 配置接入流程
    * 加密认证环节
    * 票据 access_token
